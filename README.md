# GitHub Profile Finder

## 📌 Project Overview

GitHub Profile Finder is a web application that allows users to search for any GitHub profile and view important account information in real time using the GitHub API. The project demonstrates API integration, asynchronous JavaScript, and dynamic content rendering.

## 🚀 Features

* Search GitHub users by username
* Fetch data using GitHub REST API
* Display profile picture
* Display name and bio
* Show followers and following count
* Show public repositories count
* Direct link to GitHub profile
* Loading state while fetching data
* Error handling for invalid usernames
* Fully responsive design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Fetch API
* GitHub REST API

## 📸 Functionality

1. Enter a GitHub username.
2. Click the Search button.
3. Application fetches user data from GitHub API.
4. Profile details are displayed instantly.
5. If the username is invalid, an error message is shown.
